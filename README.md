# BIBLIOTECA DIGITAL

Desarrollo de un sistema de gestión de biblioteca digital en Python, aplicando Programación Orientada a Objetos avanzada y buenas prácticas de diseño.

**Autor:** Christian Cabo Terriza

---

## Instalación

Clonar el repositorio:

```bash
git clone https://github.com/CCTChristianGit/Biblioteca_digital_Christian.git
```

```bash
cd Biblioteca_digital_Christian
```

Instalar las dependencias necesarias:

```bash
pip install -r requirements.txt
```

---

## Ejecutar programa

```bash
python main.py
```

---

## Ejecutar tests

```bash
pytest
```
